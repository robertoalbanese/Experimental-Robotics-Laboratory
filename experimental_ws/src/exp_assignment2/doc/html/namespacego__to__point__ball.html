def go_to_point_ball.main ( )

Initialization of the node.

This function initialize some of the publisher and subscriber executed by the program:

  • A publisher to the topic "/ball/cmd_vel" to send command velocities to the ball.
  • A publisher to the topic "/gazebo/set_link_state" to update the current link state of the ball.
  • A subscriber to the topic "/ball/odom" to read the current position of the ball.
  • An action server to the server '/robot/reaching_goal' to drive the ball in a new position.
